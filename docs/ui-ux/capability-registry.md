## Explicit deferrals

- Google Ads mutation execution is deferred; current Google RSA support remains generation/export only.
- Meta targeting dimensions broader than country and age are deferred. Requests containing broader targeting are rejected rather than silently narrowed.
- SMTP and additional native outbound providers are deferred; the provider seam is implemented with Resend only.
- Mobile visual QA is deferred because the supported product target is the desktop application at 1024px and wider.
- Batch content approval is deferred. Batch authorization acts only on already-proposed external actions and never replaces draft approval.
- Distributed queue infrastructure is deferred. Current bounded workers and resumable batch ledgers remain the deliberate single-process implementation.
